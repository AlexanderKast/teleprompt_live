Generador de comentarios
Agrega tu API key en Stream Settings → API Keys. En Prompter, escribe el tema del live y presiona "Generar comentarios con IA". Los comentarios se insertan automáticamente con timestamps correctos.

Analizador de guion
Con tu guion listo, presiona "Analizar guion con IA". Recibirás una puntuación del 1 al 10, fortalezas, zonas de riesgo y sugerencias específicas.

Metadata con IA
En Metadata, usa los botones "✨ IA" para generar automáticamente la descripción y los hashtags optimizados para tu plataforma.

Clave de Gemini gratuita
Ve a aistudio.google.com → "Get API Key" → crea una clave gratuita → pégala en Stream Settings → API Keys.

  • 🎯 Coloca tu primer [COMENTARIO] antes del minuto 2. La audiencia necesita ver actividad desde el inicio.
  • ⚡ Usa [EMOCION] URGENCIA solo 1 o 2 veces por live. Si lo abusas, pierde impacto.
  • 🔄 Alterna entre [GUION], [ACCION] y [COMENTARIO] cada 45-60 segundos para mantener dinamismo.
  • 💬 Los mejores comentarios son los de objeción. "No tengo dinero", "No sé nada de esto" — respóndelos en el live.
  • ⏱ Un [COUNTDOWN] en el minuto 1 aumenta la retención inicial. La gente se queda a ver qué pasa.
  • 🎭 El bloque [EMOCION] REFLEXION es poderoso justo antes de tu llamada a la acción.
  • 📱 Ensaya el live completo en modo privado al menos una vez antes de grabar el fake live final.
  • 🏆 El objetivo no es la perfección — es que parezca real. Los errores pequeños ayudan a la autenticidad.

Cómo funciona
Lee automáticamente todos los bloques [COMENTARIO] de tu guion y los prepara para LiveCake/Pancake con avatares reales. También puedes añadir comentarios de relleno activando los checkboxes.

Paso 1 — Agrega tu Gemini API Key
Ve a Stream Settings → API Keys y asegúrate de tener tu Gemini key. La misma clave se usa para generar avatares realistas con Gemini Imagen 3. Sin key, usa DiceBear gratis.

Paso 2 — Genera avatares
En LiveCake Export haz clic en "🎨 Generar avatares". Se crea una foto de perfil realista para cada usuario del guion. Verás el progreso en tiempo real.

Paso 3 — Descarga el CSV
Haz clic en "⬇️ Descargar CSV para LiveCake". Tiene 4 columnas: Title (username), Content (comentario), Time (tiempo) e Image (avatar URL). Súbelo a LiveCake/Pancake.

Modo gratis (sin key)
Si no configuras Gemini key, la app genera avatares de DiceBear automáticamente. Para persistencia entre sesiones, configura Supabase Storage en las mismas opciones.
